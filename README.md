# Firework Wars Items Resource Pack
Resource pack for Firework Wars items

## How to Use
If you don't have the `.zip` file yet, you can download it from the releases page on the right.
<br><br>
After downloading the `.zip` file, move it to your resource pack folder in-game.
<br><br>
If you don't know where this folder is, one way of opening it is to launch Minecraft, and click `Options > Resource Pack > Open Pack Folder`.
<br><br>
Now it should appear in the "Available" section of your resource packs. To use it, click the > button on the pack to move it into "Selected".
